Musixmatch Power Enhancer

The ultimate Musixmatch tool and Tampermonkey userscript that supercharges your mxm experience. As a powerful enhancer mxm, it unlocks Spotify source IDs, track metadata badges, and seamless Curators shortcuts.

Musixmatch Enhancer Core Features

🎧 Connected Sources Instantly view and copy Spotify, Apple Music, and Amazon Music IDs linked to albums, artists, and tracks.
📊 Enhanced Metadata See exact release dates, ISRCs, common track IDs (Abstrack), and clear visual badges for lyrics and structure.
🛠️ Curators Tools Drafts manager, published state locks, and ultra-fast navigation shortcuts embedded directly into the Curators interface.
🗺️ Availability Maps One-click access to Spotify Availability Map to quickly verify region locks for any track or album.
